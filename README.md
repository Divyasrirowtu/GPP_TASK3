# Custom Columnar File Format (CCF)

This project implements a simplified analytical **columnar file format**, similar to Parquet/ORC.  
It includes:

- Binary writer (`writer.py`)
- Binary reader (`reader.py`)
- Compression support (optional)
- Column-wise storage
- Selective column reads
- Test files (`test_sample.ccf`)
- Round-trip testing (`roundtrip_test.py`)
